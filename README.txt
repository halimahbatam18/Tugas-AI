# Project AI - Tugas UTS

Proyek ini merupakan tugas UTS mata kuliah **Artificial Intelligence & Big Data**.

## Deskripsi Proyek
Proyek ini membuat model **Machine Learning sederhana** menggunakan algoritma **Logistic Regression** untuk memprediksi hasil berdasarkan dua variabel:
- Usia  
- Gaji  

Model ini digunakan untuk melatih dan menguji data sederhana, kemudian menampilkan hasil akurasi serta prediksi model.

## File dalam Repository
- `projectAI.ipynb` → Notebook utama yang berisi kode model AI.  
- `PERFORMA MODEL.jpeg` → Screenshot hasil akurasi, confusion matrix, dan laporan klasifikasi.  
- `HASIL PREDIKSI MODEL.jpeg` → Screenshot hasil prediksi dari model.  
- `README.txt` → Penjelasan singkat tentang proyek ini.

## Hasil Model
- **Akurasi Model:** 1.0  
- Model berhasil memprediksi semua data uji dengan benar (100% akurat).  
- Hasil prediksi model: `[0, 1, 0]`

## Kesimpulan
Model Logistic Regression dapat bekerja dengan sangat baik pada dataset sederhana. Proyek ini menunjukkan dasar pemahaman tentang cara kerja model prediksi menggunakan algoritma supervised learning.
